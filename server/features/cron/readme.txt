Cron задачи для Express сервера

Структура:
- interfaces.ts - интерфейсы для cron задач
- register.ts - функция регистрации задач
- jobs/ - директория с задачами
- index.ts - экспорт всех задач

Как создать новую задачу:

1. Создайте файл в директории jobs/ (например: jobs/my-task.ts)
2. Определите задачу с интерфейсом ICronJob:

import type { ICronJob } from '../interfaces';

export const myTask: ICronJob = {
    name: 'my-task',
    schedule: '0 0 0 * * *', // каждый день в полночь
    enabled: true,
    handler: async () => {
        // ваша логика здесь
    },
};

3. Добавьте задачу в массив cronJobs в index.ts:

import { myTask } from './jobs/my-task';
export const cronJobs = [
    exampleCronJob,
    myTask,
];

Примеры cron выражений (с секундами):

'*/5 * * * * *'     - каждые 5 секунд
'*/30 * * * * *'    - каждые 30 секунд
'0 */5 * * * *'     - каждые 5 минут
'0 0 * * * *'       - каждый час
'0 0 0 * * *'       - каждый день в полночь
'0 0 0 * * 0'       - каждое воскресенье в полночь
'0 0 9 * * 1-5'     - каждый рабочий день в 9:00
'0 0 0 1 * *'       - первого числа каждого месяца
'0 0 0 1 1 *'       - 1 января в полночь
'0 30 14 * * *'     - каждый день в 14:30:00
'* */1 * * * *'     - каждую секунду каждой минуты каждого часа

Формат: секунды минуты часы день месяц день_недели

Дни недели: 0 = воскресенье, 1 = понедельник, ..., 6 = суббота
